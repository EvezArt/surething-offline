# Corrections

## E8
- E8 root system has 240 roots in R^8.
- 248 is the dimension of the E8 Lie algebra, not root count.
- Hash functions should not be treated as generating E8 roots.
- Use exact canonical root set and verify via Gram matrix and closure.

## CPF
- CPF is a composite experimental metric, not a definition of qualia.
- It may combine integration and prediction-error measures.
- It should be evaluated empirically, not interpreted as consciousness detection.
